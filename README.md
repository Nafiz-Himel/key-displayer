# Key Displayer ⌨️

A minimal, clean, and real-time keyboard key detector built using HTML, CSS, and vanilla JavaScript. Whenever a user presses any key on their keyboard, it instantly displays the key name in giant typography on the screen.

## ✨ Features
* **Real-time Detection:** Captures and displays any keypress instantly.
* **Minimalist UI:** Sleek, dark-themed interface centered around the key display.
* **Responsive Design:** Fully centered and scales perfectly on all screen sizes.
* **No Frameworks:** Purely written in HTML5, CSS3, and modern JavaScript (ES6).
